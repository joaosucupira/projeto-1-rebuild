# projeto-1-rebuild
 Repositório voltado para a repetição dos projetos da disciplina de fundamentos de programação lecionada no curso de sistemas de informação da Universidade Tecnológica Federal do Paraná.

 A intenção desse repositório é demonstrar como estou resolvendo os problemas propostos desde o ínicio do semestre, agora,
 com as ferramentas do aprendizado completo. Como sendo o primeiro projeto da disciplina, o trabalho 1 requer uma solução
 mais eficiente conforme consigo utlizar o conceitos de structs, vetores e outros artifícios tanto da sintaxe C quanto dos
 algoritmos que foram praticados ao longo das aulas.
